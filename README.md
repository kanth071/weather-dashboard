# 🌤️ Weather Dashboard

A clean and responsive Weather Dashboard web application that provides real-time weather information for any city using a weather API.

## 🚀 Features

- Search weather by city name
- Displays temperature, humidity, and wind speed
- Weather condition with icon
- Responsive and user-friendly interface

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)  
- Weather API (e.g., OpenWeatherMap)
